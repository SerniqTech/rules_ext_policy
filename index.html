Privacy Policy for New Grid Ruler

Last updated: 27 Dec 2025

Overview

New Grid Ruler is a browser extension designed to help users create grid guides and measure distances directly on web pages. The extension works entirely within the user's browser.

Data Collection

New Grid Ruler does not collect, store, transmit, or share any personal or sensitive user data.

How the Extension Works

All functionality runs locally in the browser. Visual guides and measurement tools are injected temporarily only when the user activates the extension.

Third-Party Services

This extension does not use third-party services, analytics, or external APIs.

Data Sharing

No user data is sold, transferred, or shared with third parties.

Changes to This Policy

Any future updates to this policy will be reflected on this page.

Contact

For questions, please contact the developer via the Chrome Web Store listing.
